## 🎯 الهدف

توفير نظام بحث وفلترة ذكي ومتقدم للوظائف والدورات التدريبية، مما يوفر الوقت ويحسن تجربة المستخدم بشكل كبير.

---

## 👥 المستخدمون المستهدفون

1. **الباحثون عن عمل** - للعثور على الوظائف المناسبة بسرعة
2. **الطلاب** - للبحث عن الدورات التدريبية
3. **الشركات** - للبحث عن المرشحين المناسبين

---

## 📝 User Stories

### 1. البحث الذكي

**كمستخدم باحث عن عمل:**
- أريد البحث عن الوظائف باستخدام كلمات مفتاحية
- أريد أن يقترح النظام كلمات بحث ذات صلة
- أريد رؤية نتائج البحث بشكل فوري (live search)
- أريد البحث في: العنوان، الوصف، المهارات، اسم الشركة

**معايير القبول:**
- [-] البحث يعمل على جميع الحقول المذكورة
- [-] النتائج تظهر خلال أقل من 500ms
- [-] الاقتراحات التلقائية تظهر بعد كتابة 3 أحرف
- [-] دعم البحث بالعربية والإنجليزية

---

### 2. الفلترة المتقدمة

**كمستخدم:**
- أريد فلترة الوظائف حسب الراتب (نطاق من-إلى)
- أريد فلترة حسب الموقع (المدينة/الدولة)
- أريد فلترة حسب نوع العمل (دوام كامل، جزئي، عن بعد، هجين)
- أريد فلترة حسب مستوى الخبرة (مبتدئ، متوسط، خبير)
- أريد فلترة حسب المهارات المطلوبة
- أريد فلترة حسب تاريخ النشر (اليوم، آخر أسبوع، آخر شهر)
- أريد فلترة حسب نوع الشركة (صغيرة، متوسطة، كبيرة)

**معايير القبول:**
- [x] جميع الفلاتر تعمل بشكل صحيح
- [x] يمكن تطبيق فلاتر متعددة في نفس الوقت
- [-] الفلاتر تحفظ في URL للمشاركة
- [-] عداد النتائج يتحدث مع كل فلتر
- [-] زر "مسح الفلاتر" يعيد كل شيء للوضع الافتراضي

---

### 3. حفظ عمليات البحث

**كمستخدم:**
- أريد حفظ عمليات البحث المفضلة لدي
- أريد إعطاء اسم لكل عملية بحث محفوظة
- أريد رؤية قائمة بعمليات البحث المحفوظة
- أريد تعديل أو حذف عمليات البحث المحفوظة
- أريد تطبيق عملية بحث محفوظة بنقرة واحدة

**معايير القبول:**
- [-] يمكن حفظ حتى 10 عمليات بحث لكل مستخدم
- [-] كل عملية بحث تحفظ: الكلمات المفتاحية + جميع الفلاتر
- [x] قائمة البحث المحفوظ تظهر في الصفحة الرئيسية
- [-] إشعار عند حفظ/تعديل/حذف عملية بحث

---

### 4. التنبيهات الذكية

**كمستخدم:**
- أريد تفعيل تنبيهات لعمليات البحث المحفوظة
- أريد استلام إشعار عند ظهور وظائف جديدة تطابق بحثي
- أريد اختيار طريقة التنبيه (إشعار داخل التطبيق، بريد إلكتروني)
- أريد تحديد تكرار التنبيهات (فوري، يومي، أسبوعي)

**معايير القبول:**
- [-] التنبيهات تعمل بشكل تلقائي عند نشر وظيفة جديدة
- [-] المستخدم يمكنه تفعيل/تعطيل التنبيهات لكل عملية بحث
- [-] الإشعارات تحتوي على رابط مباشر للوظيفة
- [-] لا يتم إرسال تنبيهات مكررة

---

### 5. البحث بالخريطة (Map View)

**كمستخدم:**
- أريد رؤية الوظائف على خريطة تفاعلية
- أريد تحديد منطقة جغرافية للبحث
- أريد رؤية عدد الوظائف في كل منطقة
- أريد النقر على علامة في الخريطة لرؤية تفاصيل الوظيفة

**معايير القبول:**
- [x] الخريطة تعرض جميع الوظائف المتاحة
- [-] يمكن رسم دائرة أو مربع للبحث في منطقة محددة
- [-] العلامات تتجمع (cluster) عند التصغير
- [-] الخريطة تدعم العربية والإنجليزية
- [-] يمكن التبديل بين عرض القائمة وعرض الخريطة

---

### 6. الفلترة المتقدمة للمهارات

**كمستخدم:**
- أريد البحث عن وظائف تتطلب مهارات محددة
- أريد استخدام منطق AND/OR للمهارات (كل المهارات أو أي مهارة)
- أريد رؤية الوظائف التي تطابق معظم مهاراتي
- أريد استبعاد وظائف تتطلب مهارات معينة

**معايير القبول:**
- [ ] يمكن اختيار مهارات متعددة من قائمة
- [ ] يمكن التبديل بين منطق AND/OR
- [ ] النتائج تُرتب حسب نسبة المطابقة
- [ ] عرض نسبة المطابقة لكل وظيفة (مثلاً: 80% مطابقة)

## 🔧 المتطلبات التقنية

### Backend

1. **Search API**
   - Endpoint: `GET /search/jobs`
   - Endpoint: `GET /search/courses`
   - دعم pagination
   - دعم sorting (حسب التاريخ، الراتب، المطابقة)

2. **Saved Searches API**
   - `POST /search/saved` - حفظ عملية بحث
   - `GET /search/saved` - جلب عمليات البحث المحفوظة
   - `PUT /search/saved/:id` - تعديل عملية بحث
   - `DELETE /search/saved/:id` - حذف عملية بحث

3. **Search Alerts API**
   - `POST /search/alerts` - تفعيل تنبيه
   - `GET /search/alerts` - جلب التنبيهات
   - `PUT /search/alerts/:id` - تعديل تنبيه
   - `DELETE /search/alerts/:id` - حذف تنبيه

4. **Database Indexes**
   - Text index على: title, description, skills
   - Geo index للمواقع الجغرافية
   - Compound indexes للفلاتر المتعددة

### Frontend

1. **Search Page**
   - شريط بحث مع autocomplete
   - لوحة فلاتر جانبية
   - عرض النتائج (قائمة/بطاقات)
   - Pagination

2. **Map View**
   - تكامل مع Google Maps أو Mapbox
   - Markers للوظائف
   - Clustering
   - Info windows

3. **Saved Searches Panel**
   - قائمة بعمليات البحث المحفوظة
   - أزرار تعديل/حذف
   - مؤشر التنبيهات

## 📊 مؤشرات الأداء (KPIs)

1. **سرعة البحث**: < 500ms
2. **معدل استخدام الفلاتر**: > 60%
3. **معدل حفظ عمليات البحث**: > 30%
4. **معدل تفعيل التنبيهات**: > 20%
5. **معدل استخدام Map View**: > 15%

3. لا يتم مشاركة بيانات البحث مع أطراف ثالثة
