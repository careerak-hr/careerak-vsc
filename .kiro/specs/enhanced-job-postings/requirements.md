# تحسينات صفحة الوظائف - المتطلبات

## 📋 معلومات الميزة

- **اسم الميزة**: تحسينات صفحة الوظائف (Enhanced Job Postings Page)
- **الأولوية**: ⭐⭐⭐⭐⭐ (عالية جداً)
- **الحالة**: قيد التخطيط

---

## 🎯 الهدف

تحسين تجربة تصفح الوظائف لتكون أكثر مرونة وجاذبية، مع إضافة ميزات تساعد الباحثين عن عمل في اتخاذ قرارات أفضل.

---

## 👥 المستخدمون المستهدفون

1. **الباحثون عن عمل** - لتصفح وحفظ الوظائف بسهولة
2. **الشركات** - لعرض وظائفهم بشكل احترافي
3. **المستخدمون الجدد** - لتجربة تصفح سلسة وسريعة

---

## 📝 User Stories

### 1. عرض Grid/List قابل للتبديل

**كباحث عن عمل:**
- أريد التبديل بين عرض Grid (بطاقات) وعرض List (قائمة)
- أريد حفظ تفضيلي للعرض
- أريد عرض Grid للتصفح السريع
- أريد عرض List لرؤية تفاصيل أكثر

**معايير القبول:**
- [-] زر تبديل بين Grid و List
- [-] عرض Grid: 3 أعمدة على Desktop، 2 على Tablet، 1 على Mobile
- [-] عرض List: صف واحد لكل وظيفة مع تفاصيل أكثر
- [-] حفظ التفضيل في localStorage
- [-] انتقال سلس بين العرضين (animation)
- [-] أيقونات واضحة للتبديل (Grid icon, List icon)

---

### 2. حفظ الوظائف المفضلة (Bookmark)

**كباحث عن عمل:**
- أريد حفظ الوظائف المهمة لمراجعتها لاحقاً
- أريد رؤية جميع الوظائف المحفوظة في صفحة واحدة
- أريد إزالة وظيفة من المفضلة بسهولة
- أريد إشعار عند تغيير حالة الوظيفة المحفوظة

**معايير القبول:**
- [-] أيقونة Bookmark (قلب أو نجمة) على كل وظيفة
- [-] تبديل الحالة بنقرة واحدة
- [x] تغيير اللون عند الحفظ (رمادي → ذهبي/أحمر)
- [x] صفحة "الوظائف المحفوظة" منفصلة
- [x] عداد للوظائف المحفوظة
- [x] إشعار عند إغلاق وظيفة محفوظة
- [x] مزامنة عبر الأجهزة (حفظ في قاعدة البيانات)

---

### 3. مشاركة الوظيفة على وسائل التواصل

**كباحث عن عمل:**
- أريد مشاركة وظيفة مع أصدقائي
- أريد نسخ رابط الوظيفة
- أريد مشاركة على WhatsApp, LinkedIn, Twitter, Facebook
- أريد معاينة جذابة عند المشاركة (Open Graph)

**معايير القبول:**
- [x] زر "مشاركة" على كل وظيفة
- [x] قائمة منبثقة بخيارات المشاركة
- [x] 5 خيارات: نسخ الرابط، WhatsApp، LinkedIn، Twitter، Facebook
- [x] نسخ الرابط تلقائياً مع رسالة تأكيد
- [-] Open Graph tags للمعاينة الجذابة
- [-] تتبع عدد المشاركات (analytics)

---

### 4. عرض "وظائف مشابهة"

**كباحث عن عمل:**
- أريد رؤية وظائف مشابهة عند عرض وظيفة
- أريد وظائف في نفس المجال
- أريد وظائف بنفس المهارات المطلوبة
- أريد وظائف في نفس المدينة

**معايير القبول:**
- [-] قسم "وظائف مشابهة" في صفحة تفاصيل الوظيفة
- [-] عرض 4-6 وظائف مشابهة
- [-] خوارزمية تشابه بناءً على: المجال، المهارات، الموقع، الراتب
- [x] نسبة التشابه (اختياري)
- [x] تحديث ديناميكي عند تغيير الوظيفة
- [-] Carousel للتمرير بين الوظائف

---

### 5. تقدير الراتب بناءً على السوق

**كباحث عن عمل:**
- أريد معرفة إذا كان الراتب المعروض عادل
- أريد مقارنة مع متوسط السوق
- أريد رؤية نطاق الرواتب للوظائف المشابهة
- أريد معرفة العوامل المؤثرة في الراتب

**كشركة:**
- أريد معرفة إذا كان راتبي تنافسي
- أريد جذب أفضل المرشحين

**معايير القبول:**
- [-] حساب متوسط الراتب للوظائف المشابهة
- [-] عرض نطاق الراتب (الأدنى - الأعلى)
- [x] مؤشر بصري (أقل من المتوسط، متوسط، أعلى من المتوسط)
- [x] ألوان: أحمر (أقل)، أصفر (متوسط)، أخضر (أعلى)
- [x] tooltip يشرح الحساب
- [-] تحديث شهري للبيانات

**مثال:**
```
الراتب المعروض: 5000 ريال
متوسط السوق: 6000 ريال
النطاق: 4500 - 7500 ريال
التقييم: أقل من المتوسط بـ 17% 🔴
```

## 🔧 المتطلبات التقنية

### Backend

1. **Bookmark Service**
   - حفظ/إزالة وظيفة من المفضلة
   - جلب الوظائف المحفوظة
   - إشعارات عند تغيير حالة الوظيفة

2. **Similar Jobs Engine**
   - خوارزمية حساب التشابه
   - تخزين مؤقت للنتائج

3. **Salary Estimation Service**
   - حساب متوسط الراتب
   - تحديث دوري للبيانات

4. **Company Info Service**
   - جلب معلومات الشركة
   - تقييمات الشركة

5. **Analytics Service**
   - تتبع المشاركات
   - تتبع المشاهدات

6. **APIs**
   - `POST /jobs/:id/bookmark` - حفظ وظيفة
   - `DELETE /jobs/:id/bookmark` - إزالة من المفضلة
   - `GET /jobs/bookmarked` - جلب الوظائف المحفوظة
   - `GET /jobs/:id/similar` - وظائف مشابهة
   - `GET /jobs/:id/salary-estimate` - تقدير الراتب
   - `GET /companies/:id/info` - معلومات الشركة
   - `POST /jobs/:id/share` - تسجيل مشاركة
   - `GET /jobs/:id/analytics` - إحصائيات الوظيفة

7. **Database Models**
   - JobBookmark (الوظائف المحفوظة)
   - JobShare (المشاركات)
   - SalaryData (بيانات الرواتب)
   - CompanyInfo (معلومات الشركات)

### Frontend

1. **View Toggle Component**
   - تبديل Grid/List
   - حفظ التفضيل

2. **Job Card Component**
   - عرض Grid
   - عرض List
   - Bookmark button
   - Share button

3. **Bookmark Page**
   - قائمة الوظائف المحفوظة
   - فلترة وبحث

4. **Share Modal**
   - خيارات المشاركة
   - نسخ الرابط

5. **Similar Jobs Section**
   - Carousel
   - بطاقات مصغرة

6. **Salary Indicator**
   - مؤشر بصري
   - Tooltip

7. **Company Card**
   - معلومات الشركة
   - تقييمات

8. **Skeleton Loader**
   - Grid skeleton
   - List skeleton

### التقنيات المقترحة

- **Frontend**: React, Tailwind CSS, React Icons, Framer Motion (animations)
- **Backend**: Node.js, Express
- **Database**: MongoDB
- **Caching**: Redis (للوظائف المشابهة)
- **Share**: Web Share API + fallback
- **Skeleton**: React Content Loader أو custom CSS

## 📊 مؤشرات الأداء (KPIs)

1. **معدل الحفظ**: > 30% من المستخدمين يحفظون وظائف
2. **معدل المشاركة**: > 10% من الوظائف تُشارك
3. **معدل النقر على الوظائف المشابهة**: > 20%
4. **رضا المستخدمين**: > 4.5/5 عن تجربة التصفح
5. **سرعة التحميل**: < 2 ثانية للصفحة الكاملة
