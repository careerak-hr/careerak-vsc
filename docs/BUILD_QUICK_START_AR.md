# دليل البناء السريع - Careerak

## البناء السريع (3 خطوات)

### 1️⃣ إصلاح مشاكل Gradle (إذا لزم الأمر)
```cmd
fix_gradle_issues.bat
```
اختر الخيار 4 للتنظيف الشامل

### 2️⃣ بناء التطبيق
```cmd
build_careerak_optimized.bat
```

### 3️⃣ انتظر حتى يكتمل البناء
- أول بناء قد يستغرق 10-15 دقيقة
- البناءات اللاحقة أسرع (2-5 دقائق)

## النتيجة

ستجد ملف APK في:
```
apk_output\Careerak-v1.0-debug.apk
```

## إذا واجهت مشاكل

راجع الدليل الكامل:
```
docs\BUILD_ISSUES_SOLUTIONS_AR.md
```

## ملفات البناء المتوفرة

| الملف | الوصف | متى تستخدمه |
|------|-------|-------------|
| `build_careerak_optimized.bat` | ⭐ موصى به - محسّن ومستقر | الاستخدام اليومي |
| `build_careerak_clean.bat` | بناء نظيف بدون تحذيرات | عند الحاجة لبناء نظيف |
| `build_careerak_final.bat` | بناء الإنتاج | للنشر النهائي |
| `fix_gradle_issues.bat` | إصلاح مشاكل Gradle | عند وجود مشاكل |

## المتطلبات

- ✅ Node.js و npm مثبتين
- ✅ Java JDK 21 مثبت
- ✅ Android SDK مثبت
- ✅ اتصال إنترنت مستقر (للبناء الأول)

## نصيحة مهمة

⚠️ **لا تقلق إذا استغرق البناء الأول وقتًا طويلاً!**

Gradle يقوم بتحميل جميع المكتبات المطلوبة في أول مرة.
البناءات التالية ستكون أسرع بكثير.
