## 🔗 التكامل

### مع نظام التوصيات
```javascript
const estimate = await salaryEstimatorService.estimateSalary(job);

if (estimate && estimate.comparison === 'above') {
  job.priority += 10; // زيادة الأولوية
}
```

### مع نظام الإشعارات
```javascript
if (estimate && estimate.comparison === 'below') {
  await notificationService.send({
    userId: user._id,
    type: 'salary_alert',
    message: `الراتب أقل من المتوسط بنسبة ${estimate.percentageDiff}%`
  });
}
```
