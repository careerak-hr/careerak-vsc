## 🚀 كيفية الاستخدام

### 1. تحديث الوظائف الموجودة (مرة واحدة)
```bash
cd backend
npm run urgent:update
```

### 2. إعداد Cron Job (دوري)
```bash
# PM2 (موصى به)
pm2 start ecosystem.config.js --only urgent-jobs-updater

# أو Linux Cron
0 */6 * * * cd /path/to/backend && npm run urgent:cron
```

### 3. إضافة expiryDate لوظيفة جديدة
```javascript
const job = new JobPosting({
  title: 'مهندس Backend',
  // ... باقي الحقول
  expiryDate: new Date(Date.now() + 5 * 24 * 60 * 60 * 1000) // 5 أيام
});

await job.save();
// isUrgent = true تلقائياً
```
