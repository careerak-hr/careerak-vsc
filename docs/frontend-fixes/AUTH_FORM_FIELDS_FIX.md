## المشاكل المكتشفة

### 1. قائمة البلدان (Country Selector)
- **المشكلة:** كانت تعرض `c.name` فقط بدون مراعاة اللغة المختارة
- **الحل:** تم تعديلها لعرض `name_ar` للعربية و `name_en` للإنجليزية
- **الكود:**
```jsx
{countries.map(c => (
  <option key={c.key} value={c.key}>
    {c.flag} {language === 'ar' ? c.name_ar : c.name_en}
  </option>
))}
```

### 2. الهينتات في القوائم المنسدلة
- **المشكلة:** الهينتات كانت options عادية يمكن اختيارها
- **الحل:** تم إضافة `disabled selected` لجميع الهينتات
- **القوائم المصلحة:**
  - ✅ `gender` (الجنس)
  - ✅ `education` (المستوى العلمي)
  - ✅ `countryCode` (كود البلد)
  - ✅ `specialNeedType` (نوع الاحتياج)
  - ✅ `industry` (مجال العمل)
  - ✅ `country` (البلد)

### 3. ألوان الهينتات
- **المشكلة:** لم يكن هناك تحديد واضح للون الهينتات
- **الحل:** تم إضافة قواعد CSS:
```css
/* لون الهينت في القوائم المنسدلة */
.auth-select-base option[disabled] {
  color: #9CA3AF !important; /* رمادي */
}

/* لون الخيارات العادية */
.auth-select-base option:not([disabled]) {
  color: #304B60 !important; /* كحلي */
}
```
