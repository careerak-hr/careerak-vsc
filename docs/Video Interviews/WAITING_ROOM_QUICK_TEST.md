# اختبار سريع لغرفة الانتظار (5 دقائق)

## 🚀 البدء السريع

### 1. تشغيل الاختبارات التلقائية (دقيقتان)

```bash
cd backend

# تثبيت التبعيات (إذا لم تكن مثبتة)
npm install

# تشغيل اختبارات غرفة الانتظار
npm test -- waitingRoom.integration.test.js

# النتيجة المتوقعة: ✅ 36/36 اختبارات نجحت
```

---

### 2. اختبار يدوي سريع (3 دقائق)

#### الخطوة 1: إنشاء مقابلة (30 ثانية)
1. افتح التطبيق كمضيف (شركة)
2. اذهب إلى "المقابلات" → "إنشاء مقابلة جديدة"
3. فعّل "غرفة الانتظار"
4. احفظ المقابلة

#### الخطوة 2: الانضمام كمرشح (30 ثانية)
1. افتح نافذة متصفح جديدة (Incognito)
2. سجل دخول كمرشح
3. افتح رابط المقابلة
4. انقر "انضم لغرفة الانتظار"

#### الخطوة 3: التحقق من الوظائف (دقيقتان)

**في نافذة المرشح**:
- [ ] ترى رسالة الترحيب
- [ ] ترى مؤقت الانتظار
- [ ] ترى موضعك في القائمة
- [ ] يمكنك اختبار الكاميرا والميكروفون
- [ ] ترى معاينة الفيديو

**في نافذة المضيف**:
- [ ] ترى المرشح في قائمة المنتظرين
- [ ] ترى اسم وصورة المرشح
- [ ] ترى وقت انتظار المرشح
- [ ] يمكنك النقر على "قبول"

**بعد القبول**:
- [ ] المرشح ينتقل تلقائياً لغرفة المقابلة
- [ ] الكاميرا والميكروفون يعملان مباشرة
- [ ] المرشح يختفي من قائمة الانتظار
